Todo COMPONENT, nada mais é que uma função que retorna JS, HTML e CSS.

defer: serve para inofrmar ao HTML que o Component será carregado após a árvore DOM ser concluída.

Components nada mais é que (soma, elemnts e styles) encapsulados!

Sempre usar o nome do arquivo (Component) com letra maiuscula / Ex: TituloDinamico.js

O método super() chama o contructor do HTML

customElements.define('titulo-dinamico', nomeDaClass) define o que está se criando com um nome específico. Todos os customElements devem ter um hifen separando.

textContent vs innetText

textContent obtém o conteúdo de todos os elementos, incluindo os elementos script e style. por outro lado, o innerText mostra apenas os elementos para "leitura humana". textContent retorna todos os elementos de um nó

getAttribute

getAttribute() retorna o valor de um argumento específico do elemento. Se o atributo não existir, o valor retornado será null ou "" (string vazia).
